== Description ==

A simple widget to add recent, popular, and random posts. This plugin is based on Jeff Star\'s sidebar tabber code. It also needs Héctor Cabrera\'s <a href=\"http://wordpress.org/extend/plugins/wordpress-popular-posts/\">WordPress Popular Posts</a> plugin installed and activated. This is the beta version of the plugin, give it a try. 

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

This plugin needs Héctor Cabrera\'s <a href=\"http://wordpress.org/extend/plugins/wordpress-popular-posts/\">WordPress Popular Posts</a>plugin installed and activated to display popular posts.
